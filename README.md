# PythonForBeginnersTestSaek
Αυτό το αποθετήριο έχει σκοπό να βοηθήσει νέους προγραμματιστές Python να κάνουν τα πρώτα τους βήματα στο GitHub.
Εδώ θα βρείτε οδηγίες, παραδείγματα και βοήθεια.

## Πώς να συνεισφέρετε:

1.  Κάντε fork αυτό το αποθετήριο.
2.  Δημιουργήστε ένα branch με το όνομά σας.
3.  Κάντε τις αλλαγές σας.
4.  Υποβάλετε ένα pull request.

Είμαστε ανοιχτοί σε όλες τις συνεισφορές!
